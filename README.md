# Cushion Backport

Backports the Cushion from the Minecraft 26.3 snapshot to older versions.

A Cushion is a small decoration you can place on any flat surface and sit on. It comes in all 16 dye colors.

## Features

- Craft a Cushion from 3 wool blocks of the same color
- Place it on top of any block and right click to sit down
- 16 colors, each with its own crafting recipe
- Recolor a Cushion in the crafting grid with any dye
- Recolor a placed Cushion by sneaking and right clicking it with a dye
- Cushions have no collision, so you can walk and place things through them
- A Cushion breaks if the block under it is removed

## Supported versions

- 1.20.1 (Fabric, Forge)
- 1.21.1 (Fabric, NeoForge)
- 26.1.2 (Fabric, NeoForge)

## Dependencies

Just Enough Items (JEI) is bundled for recipe viewing.
